# 素数与合数

## 概念

对任意正整数 $n>1$, 若除 $1$ 与它本身之外, $n$ 没有其它的因数, 那么 $n$ 为素数, 也称质数, 否则 $n$ 为合数. 特别的, $1$ 既不是素数也不是合数.  
素数与合数有以下的一些性质.  

!!! note 性质
    1. 设 $n$ 为大于 $1$ 的正整数, $p$ 是 $n$ 的大于 $1$ 的因数中最小的正整数, 则 $p$ 一定为素数.
    2. 若对于任意 $1$ 到 $\sqrt{n}$ 之间的素数 $p$, 都有 $p \nmid n$, 则 $n$ 为素数 $(n > 1)$.
    3. 素数有无穷多个.
